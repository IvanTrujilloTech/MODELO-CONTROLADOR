🚀 Getting Started
Prerequisites:

Docker and Docker Compose
Node.js and npm
Git
Installation Steps:

Navigate to the project directory: cd laravel-financial-app
Run containers: docker-compose up -d
Install dependencies: docker exec -it laravel-app composer install
Run migrations/seeders: docker exec -it laravel-app php artisan migrate --seed
Compile assets: npm install && npm run dev
Access: http://localhost:8000
